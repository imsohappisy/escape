SYSTEM ARCHIVE / TIPS

[ 조작법 BASIC CONTROLS ]

  • 이동: W, A, S, D 또는 방향키
  • 고속 대쉬 (회피기): SPACE BAR (4초 쿨타임 / 사용 중 완전 무적 및 강풍 무시)
  • 일시정지: ESC 또는 P

[ 게임 모드 GAME MODES ]

  • NORMAL CHRONICLE : 순수하게 30종의 공격 패턴만 등장 (최대 57기 제한).
  • HARDCORE PROTOCOL : 화면에 최대 70기의 탄막이 쏟아지며, 시야 가림이나 둔화 등의 방해 기믹 3종이 추가로 발생.

[ 방해 기믹 HAZARDS ] (하드 모드 전용 / 대미지 없음)

  • ■ 중력장 (Gravity Field) : 거대하게 퍼져나가는 보라색 파동. 내부 진입 시 이동 속도가 극도로 느려짐.
  • ■ 강풍 (Wind) : 화면을 가로지르는 흰색 기류. 플레이어를 기류 방향으로 강제로 밀어냄.
  • ■ 안개 (Fog) : 짙은 어둠이 화면을 덮음. 플레이어 주변의 아주 좁은 시야 구역만 확보됨.

[ 1급 위험: 극한 패턴 EXTREME PATTERNS ]

  • ● 위성 (Orbit Player) : 플레이어 주변을 포위하듯 나타나 궤도를 좁혀옴. 가만히 있거나 대쉬로 빠져나가야 함.
  • ■ 장벽 (Wall) : 위에서 쏟아지는 거대한 벽. 단 하나의 좁은 빈틈만 존재함.
  • ▲ 삼각 그물 (Triangle Net) : 3방향에서 화면보다 큰 빨간 삼각형 그물이 덮쳐옴.
  • ■ 쌍둥이 레이저 (Twin Laser) : 위아래를 가로지르며 압박해 들어오는 거대한 두 줄기 붉은색 막대.
  • ○ 무작위 폭탄 (Random Bombs) : 화면 곳곳에 무작위로 투하되어 천천히 옥죄어오는 초대형 라이트 그린 링.
  • ● 역결 소용돌이 (Spiral Reverse) : 회전 궤도 방향이 갑자기 반대로 꺾이는 파란색 구체.
  • ● 뱀 (Snake) : 일렬로 늘어서 물결치듯 뱀처럼 꼬리물고 날아오는 녹색 구체 편대.
  • ■ 십자 포화 (Crossfire) : 네 모서리에서 중앙으로 나란히 대각선 교차 사격되는 주황색 사각형.
  • ○ 고동치는 링 (Pulse Ring) : 심장 박동처럼 크기가 불규칙하게 커졌다 작아지는 보라색 폭발 링.
  • ＋ 스타버스트 (Starburst) : 무작위 허공에서 십자 형태로 한 번에 터져 흩어지는 파편.

[ 2급 위험: 변칙 패턴 MODIFIED PATTERNS ]

  • ▲ 스웜 (Swarm) : 플레이어를 향해 꿀벌 떼처럼 일제히 쇄도.
  • ■ 레이저 넷 (Laser Grid) : 가로 또는 세로축을 완전히 가로지르는 촘촘한 하늘색 사각형 그물.
  • ○ 우주 지뢰 (Mines) : 플레이어와 먼 곳에 무작위로 생성되어 천천히 주변을 위협하는 덩치 큰 주황색 링.
  • ▲ 포위망 (Pincer) : 양쪽 끝에서 중앙을 향해 동시에 어긋나며 좁혀오는 보라색 압박 형태.
  • ● 파동 곡선 (Sine Wave) : 부드러운 사인(Sine)파 곡선을 그리며 위아래로 춤추듯 다가오는 민트색 구체.
  • ＋ 회전 십자 (Spinner) : 정중앙에서 생성된 뒤 십자가 형태로 빙글빙글 돌며 빠르게 궤도를 넓힘.
  • ▲ 블랙홀 (Blackhole) : 맵 임의의 지점에서 외부를 향해 빨아들이듯 생성.
  • ▲ V포메이션 (V-Shape) : 양방향 V자로 꺾어지며 기만하는 주황색 삼각형.
  • ● 저격수 (Sniper) : 플레이어의 현재 높낮이를 정확히 1열 정렬하여 매우 빠르게 돌진.
  • ○ 꽃잎 (Flower) : 중앙에서 회전하며 아름답게 흩어지는 소형 노란색 링.
